About this Resume
This resume was created with the intent to test out upcoming features of HTML5 related standards -- primarily the canvas element. With that in mind, this website will not function in web browsers which do not have an implementation of such standards. Furthermore, certain effects may cause a noticable slow down in browsers that do not have a modern JavaScript engine. If this is the case, please feel free to download the PDF version of this resume.
If you are interested in the implementation of this website, the source code is licensed under the MIT license and can be obtained at github. Currently supported and tested browsers include the latest versions of: Chrome and Chromium, Firefox, Safari, Opera and IE9. Based on computer specifications performance may vary.
