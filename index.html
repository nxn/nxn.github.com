YAWS: Yet Another Web Sudoku

The aim of this project is to create a modern, fully client-side, sudoku game with robust features such as customizable keyboard controls, saving and sharing game states, hints, and more. YAWS is currently in an early stage of development.


Demo: yaws.nxn.io

Project Repository: Github

Waffle-Iron: Rust Sudoku Generator & Solver

Waffle-Iron is a sudoku puzzle generator and solver written in Rust. It's capable of being compiled into WebAssembly and executed from within a browser, where it offers consistant and improved performance in comparison to similar JavaScript implementations.


Project Repository: Github
